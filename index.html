Welcome to live.xsockets.net Example 3

This is a simple chat application using the Windows Azure hosted XSockets.NET Realtime Framework.

It uses the publish and subscribe features of XSockets.NET's JavaScript API. In addition to the Realtime communication you got a simple storage that helps you persist data between sessions etc.

Futher information about XSockets.NET AzureService live.xsockets.net can be found on http://live.xsockets.net
